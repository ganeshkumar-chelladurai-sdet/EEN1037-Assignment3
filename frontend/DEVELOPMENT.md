# React + Vite frontend

## Requirements

* NodeJS LTS v24+
* Visual Studio Code
* VS Code extension: Vitest Explorer (`vitest.explorer`)
* Chrome extension: React Developer Tools


# Running

To run the project:

* Start the [backend](../backend/DEVELOPMENT.md)

* Open this folder in Visual Studio code
* In the command prompt:
    * `npm install`
* "Run and Debug"
  * `npm run dev`
  * `Launch Chrome`
  
* Open <http://localhost:5173> in the web browser.


## Testing

In the terminal:

* `npm test`
* `npx vitest run --coverage`

In Visual Studio Code:

* Testing
  * Run Tests
  * Debug Tests
  * Run Tests With Coverage


## Source Template

This project was created from:

```
npm create vite@latest frontend -- --template react
```
