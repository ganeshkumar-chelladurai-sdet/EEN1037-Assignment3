# AGENTS.md

This project is a Django REST API backend for the associated React frontend in ../frontend.

* Use the existing `.venv` virtual environment for all project-specific Python commands. Do not create a new virtual environment.
  * `.venv/bin/python manage.py <command>`
  * `.venv/bin/pytest`    
  * `.venv/bin/pip <command>`

See [DEVELOPMENT.md](DEVELOPMENT.md) for detailed setup, migrations, seed data, and testing instructions.
